## Setup

1. Clone the repo

   ```bash
   git clone https://github.com/kymibuilds/tempori.git
   cd tempori

   ```

2. Install Dependencies

   ```bash
   npm install

   ```

3. add enviroment variables

4. Run using
   ```bash
   npm run dev
   ```
